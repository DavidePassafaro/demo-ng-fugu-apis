File System Access API

Open, read, create and save files and directories on the local filesystem via showOpenFilePicker(), showSaveFilePicker() and showDirectoryPicker().

@if (!isSupported) {
⚠️ API not available. This browser does not support the File System Access API. Use Chrome or Edge 86+ on desktop.
} @else {
✓ API available. showOpenFilePicker, showSaveFilePicker and showDirectoryPicker are all supported.

Open & read a file showOpenFilePicker() · handle.getFile() · file.text()

Create & save a file showSaveFilePicker() · handle.createWritable() · writable.write()

Browse a directory showDirectoryPicker() · for await (const [name, handle] of dirHandle)

}
